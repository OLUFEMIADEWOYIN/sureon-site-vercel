
# SureOn Power - Static Website (Ready for Vercel)

This repository contains a simple static HTML/CSS/JS website for SureOn Power.
To deploy on Vercel:
1. Create a GitHub repo and push these files (root should contain index.html).
2. Sign in to https://vercel.com with GitHub and import the repo.
